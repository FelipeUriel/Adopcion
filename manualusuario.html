Ayuda

El "Manual de Usuario de Little Pet" describe el proceso de registro en la plataforma, tanto para adoptantes como para mascotas. Los usuarios pueden completar perfiles con información personal, descripciones y fotos. La aplicación permite la navegación a través de una pantalla principal con botones de búsqueda para filtrar perfiles de mascotas.
